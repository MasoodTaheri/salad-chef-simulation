# salad chef simulation
 a “programmer-art” version of a salad chef simulation in Unity


Powers

    Speed - Increases speed of player.
    Time - Increases time of player.
    Points - Gives bonus points to the player.

Controls
Player 1

    [Arrow keys] Movement
    [P] pick up/down

Player 2

    [WASD] Movement
    [Q] Pick up/down
